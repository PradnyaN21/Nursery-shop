Online Plant Nursery Website

📌 Project Description

The Online Plant Selling Website is an e-commerce web application developed using HTML, CSS, and PHP. This platform allows users to explore and purchase a variety of indoor and outdoor plants from the comfort of their homes.

The main objective of this project is to provide an easy and convenient way for users to buy plants online while promoting a green and eco-friendly lifestyle.

🚀 Features

🌿 Browse indoor and outdoor plants
🔍 Search and filter products
🛒 Add to cart functionality
💳 Simple checkout process
👤 User-friendly interface
📱 Responsive design for mobile and desktop


🛠️ Technologies Used
Frontend: HTML, CSS
Backend: PHP
Database: MySQL
